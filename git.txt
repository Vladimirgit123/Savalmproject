$git init-инициализирует репозиторий

$git config --global user.name "Vladimir"-установка имени

$git config --global user.email"email@"-установка мыла

$git config --list-вывод настроек конфигурации

$git help (название команды)-смотрим help

$git add .-добавляет файлы проекта на индексацию,берет под наблюдение

$git commit -a -m"tochkaone"-создаем первую точку востановления
-a-добавить все файлы
-m-название точки восстановления

git remote add origin https://github.com/Vladimirgit123/Savalmproject.git
привязка локального репозитория к глобальному (сервер)
git push -u origin master
выгрузка с локального репозитория в глобальный
